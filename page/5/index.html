关于速写

2015-11-12
0 条评论

速写是一种快速的写生方法，即用简单的笔触描摹对象的造型。
快速是它的特点，但也不是一味的求快，要根据描写的对象而定。
速写不但是造型艺术的基础，也是画家创作中积累素材的好办法。
速写既能快速记录素材，也能提高我们的观察能力和表现能力，还能提高我们的默写能力。
速写分静态速写和动态速写，两者有很大的不同。
静态速写的对象往往是相对静止的，比如景物、房子、坐着、站着的人或摆着睥物品等，能给我们一定的时间描绘，这样的速写是较容易的。
动态速写所面对的对象是随时变化的。比如跳舞的人、跑动中或打斗中的人、飞驰的汽车等，根本没有反复观察的时间，仅仅依靠你对对象的动作变化过程中动态的感受和记忆。
“选定”是动态速写我有的技术。选定是指画者对活动的对象瞬间动作的特定选择，去给绘制最有造型意义的瞬间动态。

父母年事已高，儿却一事无成

2015-11-03
0 条评论

下行思思在QQ问我回去了没？“没回去啊”我说。然后她说我妈妈病了。我问是什么情况？她说是腿有问题。我心中就纳闷了，怎么就腿有问题了。记得十一回家的时候，妈妈有说过，之间由于被车子带走附带撞了下，应该是胸腔和腰椎有很大的影响，导致不能走路。让她去医院看，硬要说没问题，已经好了。可怜上一代父母。为了子女也自己看病的钱都要省了。也只能怪自己还不快点结婚，了了妈妈的一桩心愿。然后我问了下思思是什么原因，她也不清楚，是她妈妈说和她讲的。要我最好打个电话回去问问。
晚上下班回家，我就打电话给妈妈。妈妈问我有什么事，我说没什么事。家里现在在种什么菜，收成怎么样？她说种的芹菜，由于价格不好，挖了很多。然后我就问她的腿是不是有问题。她一直追问是谁跟我说的。是六阿姨？是思思？是刘松妈妈？然后又说到是刘松？婷婷？都被我掩饰过去了。我就问，是不是有什么问题？不能走路了还是怎么了。妈妈说没什么问题，就是十一我走了后先去地里干活，然后再去外婆家给外婆洗衣服。由于劳累过度，加上之前被车子弄了下，然后就站不起来了。我心想，妈妈上次去看，肯定没彻底看好。而且年龄也大了，怎么能这样劳累。洗衣服难道就不能用洗衣机吗？舅妈连洗衣机都不让外婆用。真是无语。
然后妈妈又谈到我身上，问我谈的朋友怎么样了。我说别人家里不同意。其实我根本就没谈，我去哪里找。一直在敷衍他们。但是不这样说，我也不知道怎么讲。妈妈说前几天还做梦，梦见我带别个姑娘回家了。我心里真不是个滋味。我怎么就这么笨呢？还没找个女朋友，我又不比别人差。我到底是怎么了。妈妈说因为做梦是带姑娘回家了，所以就谈不成了，梦里和现实是反着的。

如何从一个疯狂的下载者成为一个优秀的学习者

2015-10-17
0 条评论

学习知识是要靠自力，而不是他力！他人的帮助只能推动学习的步伐，但代替不了学习的全部过程。
这个世界上没有免费的午餐。
当我们为了下载大量的免费资料而沾沾自喜的时候，当我们不停地去寻找更多的资料的时候，我们已经失去了最宝贵的东西。—— 时间和生命。人生能有几回搏。
我们浪费掉一个小时，我们的生命中就失去一个小时。本来我们一生中就做了很多无意义的事而浪费时间，那么学习上就不要浪费了。与其在下载中自我陶醉，还不如多下点苦功夫去认真背点东西来得实际。
找到适合自己的学习方法，相信自己“我行我一定能做到”，关且坚持。把自己从一个疯狂的下载者变成一个优秀的学习者。
聪明的人总是故意把自己想知道的问题错误的说出来，借以抛砖引玉，得到别人正确的纠正！同时也是你自己学习的时候，提高的时候。

良哥给我讲HTML和CSS

2015-09-22
0 条评论

早上九点四十多，良哥来到公司，从我身边走过，顺便找我要了培训课件的效果图。几个星期前我就做好了，只是没有完成HTML页面的布局。过了一会，用邮箱发给他了。
下午四点多，他把页面做好发给张云勇后（他应该在忙其它的，所以才到下午才开始做的），跟我说：“等下我跟你讲下CSS，其实那东西很简单。我跟你一讲就明白，你抽空的时候可以学一学。对你以后也会用得到。”
“好的。”我回答。
差不多四点半的时候，他让我去他那边，跟我讲一下HTML和CSS。
首先跟我讲了下，布局这样的页面很容易。然后跟我讲用记事本就可以做了。然后新建了一个HTML文件。讲了下HTML的结构。然后说这样的界面就是用div布局。说白了就是div套用div.
然后用div先布局出大体框架，然后给每一个添加细节。在布局的同时给每一个div添加样式，先是在控件上添加样式。目地是告诉我这些样式是怎么操作控件的。margin和padding的使用，字休颜色，字号，背景颜色等。
他讲得很详细，我听得也很认真。虽然很多东西我略有了解。
他边讲边操作，算得上是给我上了一课。讲完后跟我讲这些东西都很简单。要我平时留意一些比较好的东西，并且收集下来，等到要用到的时候可以不用自己重新去写。而且提高了工作效率。还给我看了下他收集的一些html文件和设计素材。
这一课让我对自己反思。自己平时也收集了一些资源和素材。但是没有整理，只是用到的时候才匆匆忙忙的去找。

朋友介绍朋友

2015-09-20
0 条评论

中午在饭店吃饭，突然张静给我打来电话。问我当初跟我说的那个女生还谈不谈了。隐约听见她妹妹在旁边说“谈就谈，不谈就不谈…”
“谈啊”。我回答。
“那你么样一点信都没有”她说。
我无言以对。我能说我最近比较忙吗？我可以说最近出差，没时间吗？
借口，满满的借口。
在做一件事之前，我总是找很多理由，或者做很多准备工作。而不去做实际的行动。绘画也是，学程序也是，谈朋友也是。现在还是。我的人生充满了为自己掩饰的借口。以搪塞自己，我还没有准备好。等我准备好了，我一定可以做得更好，更成功。
可以，到现在已经有两个朋月了，我准备好了吗？我在干什么呢？
天天无休止的玩游戏，或者看动画片。最近两年来玩英雄联盟玩了这么久。遗失的是我的光阴，错过的是自己的爱情。留下的是一事无成。
我这个年龄，正常的轨迹是一个企业中的管理者了。至少学有所成，有一技之长。然后，我把自己的时间和精力都浪费到无用的事情上了。
怎么样，最次也得有一个女朋友吧。我没有。长这么大，我没有。我是一个彻底的失败者。学习，事业，爱情。什么都没留下。
她和我讲要给我介绍女朋友还是端午的时候。到现在这么久了。我却什么也没有做。我是一个男生，一个男人，我应该主动都对。然后我去像个缩头乌龟一样，躲在角落里，不敢去面对。不是不想，是不知道怎么去面对。也不知道怎么去和对方勾通。彼此的不了解，人生观，价值观都不知道。
我像一个孩子一样，还在浑浑噩噩的过着日子。
